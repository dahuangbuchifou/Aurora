# Aurora Intelligence Platform

> 个人认知增强与知识操作系统  
> Personal Cognitive Augmentation & Knowledge Operating System

Aurora 用于从多种渠道采集信息，区分事实、数据、观点与预测，形成结构化知识、个人观点和可复用输出，并为 WQRS、AI Research 等应用提供统一知识底座。

## 当前阶段

```text
多来源输入
→ 内容解析
→ 事实与观点提取
→ 结构化知识
→ 正反观点比较
→ 个人观点草案
→ Markdown输出
```

## 项目原则

1. 不把摘要当知识。
2. 不把观点当事实。
3. 不把共识当真相。
4. 不把 AI 生成内容当证据。
5. 所有重要结论保留来源、时间和证据链。
6. 先个人可用，再平台化。
7. 先跑通认知闭环，再扩大数据来源。

> Aurora 是知识与认知辅助系统，不替代投资、法律、医疗或其他专业决策。
