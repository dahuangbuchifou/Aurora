## 5. 功能验收

### DataPoint

- [ ] V1.1 DataPoint 支持 MeasurementContext；
- [ ] monetary 数据可表达 currency 和 scale；
- [ ] 可表达 CAS/IFRS；
- [ ] 可表达归属和合并范围；
- [ ] V1.0 DataPoint 可读取；
- [ ] 不自动猜测缺失口径。

### Claim

- [ ] claim_dimension 与 claim_type 同时存在；
- [ ] 默认值为 general；
- [ ] V1.0 Claim 可读取；
- [ ] 原子性 Linter 只警告，不阻止保存；
- [ ] 估值谨慎与增长预期不被自动标为直接反驳。

### Provenance

- [ ] 支持 DerivationLink；
- [ ] origin_object_ids 继续可用；
- [ ] V1.0 来源映射为 generic derived_from；
- [ ] PR → 年报可表达 summarizes；
- [ ] 计算数据可表达 calculated_from；
- [ ] Traceability 可沿两种字段回溯。

### Evidence

- [ ] 同一 independence_group 只计一个独立证据；
- [ ] 多个来源对象仍全部保留；
- [ ] support/refute/qualify/context/counterexample 规则有测试；
- [ ] 去重不跨 target 或 role。

### Traceability

- [ ] dependency=true 缺失时报 dangling；
- [ ] dependency=false 缺失不报 dangling error；
- [ ] processing_run_id 缺失不污染认知链；
- [ ] 三个 Source 回溯保持完整。
