# M2-003A 大黄裁断表 V1.0

只需要裁断存在语义选择空间的事项。

## A-001：管理层未来意图如何表达

原文：

```text
管理层表示将继续推进产能建设
```

### 方案A：Prediction

```text
statement=公司将继续推进产能建设
claim_type=prediction
asserted_by=中芯国际
time_horizon=unknown future
```

优点：直接表达底层主张，不会把未来意图误当事实。

### 方案B：Speech-act Fact Claim

```text
statement=管理层曾表示将继续推进产能建设
claim_type=fact_claim
```

优点：只确认“发生过该表述”，不确认未来会实施。

### 大G建议

采用双层语义：

1. Speech-act fact_claim：管理层作出该表述；
2. Embedded prediction：公司将继续推进产能建设。

M2-003A当前若只允许一个Claim，优先采用 Prediction。
