# 中芯国际三来源对象模型验证简报

> **用途**：Aurora M1-002 数据模型与追溯链测试  
> **数据截止日**：2026-07-12  
> **重要说明**：本文完全基于仓库提供的测试素材，未进行独立外部事实核验，不构成投资、交易或仓位建议。

## 1. 已确认的对象链

```text
网页PR / 视频转写 / 年报摘录
→ ContentUnit
→ Claim / DataPoint
→ Evidence / Fact
→ KnowledgeObject
→ Insight
→ PersonalOpinion Draft
→ OutputArtifact
```

## 2. 测试素材中的事实层

年报测试摘录提供了2025年营业收入、净利润、毛利率、资本开支、出货量、制程结构和产能利用率等数据。每项事实均通过 Evidence 回溯至具体页码和 ContentUnit。

Case A 官网PR中的重复数据没有被视为第二份独立证据；其 Evidence 与 Case C 使用相同 `independence_group`，并通过 Provenance 指向年报数据点。

## 3. 主张与观点层

- 管理层材料提出2026年收入增长、毛利率和产线量产目标。这些属于预测型 Claim，不是已经发生的 Fact。
- 视频作者认可长期国产替代逻辑，但对特定价格下的估值、成熟制程竞争、资本开支和出口管制风险更谨慎。
- “公司可能增长”与“当前价格吸引力不足”属于不同分析维度，不能简单标记为直接反驳。

## 4. 显式冲突

PR测试素材中的“部分8英寸产线利用率低于70%”与年报测试摘录的76.5%不一致。Aurora将前者保留为`disputed Claim`，并用年报页码证据建立`refute`关系，而不是静默覆盖。

## 5. 洞察

测试链路形成的 Insight 是：经营增长展望与短期估值谨慎可以同时成立。该推断保留支持对象、限制性证据、推理步骤和替代解释。

## 6. 个人观点草案

系统生成的 PersonalOpinion 保持 `draft` 且 `confirmed_by_user=false`。它只是一份待用户审核的研究草案，不代表用户已经认可，也不输出买卖或仓位结论。

## 7. 模型问题

最重要的问题是 DataPoint 的财务口径不足，以及 Claim 缺少分析维度。上述问题进入 M1-003 评审，本期不修改核心 Schema。
